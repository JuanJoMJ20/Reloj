# Reloj de Escritorio (Python + Tkinter)

Este es un proyecto educativo desarrollado para un taller universitario. Consiste en una aplicación de reloj de escritorio que soporta vistas analógicas y digitales, con múltiples modelos de diseño.

## Características
- **Reloj Analógico**: Incluye 3 modelos (Rolex, Cartier y Deportivo).
- **Reloj Digital**: Diseño moderno con estilo de pantalla LED azul.
- **Intercambiable**: Switch para alternar entre analógico y digital.
- **Navegación**: Flechas para cambiar el modelo del reloj analógico.
- **Diseño Premium**: Uso de tipografías elegantes y colores coordinados.

## Requisitos
- Python 3.x
- Librería Tkinter (incluida por defecto en la mayoría de las instalaciones de Python)

## Cómo ejecutar
Ejecuta el siguiente comando en tu terminal:
```bash
python clock_app.py
```

## Créditos
Desarrollado como parte de un taller de programación en Python.
